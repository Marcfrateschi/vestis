# VESTIS

AI-powered wardrobe and personal styling app.

## Features
- Smart wardrobe management with AI photo analysis
- AI personal stylist
- Virtual try-on (Fashn.ai integration)
- Intelligent trip packing planner

## Tech stack
- React 18 + Vite
- Supabase (auth + database + storage)
- Anthropic Claude API (vision + styling)
- Fashn.ai (virtual try-on)

## Environment variables
Set these in Vercel:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `ANTHROPIC_API_KEY`
